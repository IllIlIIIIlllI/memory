Wie spielt man Memory?


Memory ist ein Konzentrationsspiel, bei dem es darum geht, Kartenpaare aufzudecken. Dabei werden alle Karten zu Beginn des Spiels mit dem Bild nach unten auf einem Spielfeld ausgelegt. Der Spieler kann nun durch Anklicken von zwei Karten versuchen, ein passendes Kartenpaar aufzudecken.
